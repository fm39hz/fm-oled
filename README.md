## Installation
* Get the source:
   * Clone this repositories
   * Get this dependencies ready:
       * [Nodejs](https://nodejs.org/en)
       * Install vsce:
        ```bash
        npm install -g @vscode/vsce
        ```
* Build:
    * Open CMD, navigate to folder contain the 'package.json', then type:
        ```bash
        vsce package
        ```
    * Install vsix file from VSCode

**Enjoy!**
